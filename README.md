# Sistema Biblioteca


## Objetivo

O sistema facilita emprestimos e gera relatorios de uso, facilitando o gerenciamento dos bibliotecários

## Requisitos

- Python instalado

## Como executar

Para iniciar o sistema, na mesma pasta do projeto, rode no terminal:

```python3 main.py```

ou se não funcionar:

```python main.py```
